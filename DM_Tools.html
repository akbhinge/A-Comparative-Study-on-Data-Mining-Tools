DATA MINING TOOLS





RAPID MINER

About

Rapid Miner is a data mining tool used to implement various classification and clustering algorithms. An important feature of Rapid Miner is its ability to display results visually. It is more powerful as compared to Weka because of language independance.

Rapid Miner also provides an integrated environment for machine learning, data mining, text mining, predictive analytics and business analytics. It is used for business and industrial applications as well as for research, education, training, rapid prototyping, and application development and supports all steps of the data mining process.

General Features

  • Rapid Miner is an environment for machine learning and data mining processes.

  • Rapid miner uses XML to describe the operator trees modelling knowledge discovery process.

  • It has flexible operators for data input and output file formats.

  • It contains more than 100 learning schemes for regression classification and clustering analysis.

  • Rapid Miner produces a selection of charts and visualizations automatically, choosing the most appropriate settings based on data properties

  • Rapid Miner supports about twenty two file formats.

  • Rapid Miner includes many learning algorithms in addition to WEKA.

  • It easily reads and writes Excel files and different databases.

  • If you set up an illegal work flows Rapid Miner suggest Quick Fixes to make it legal.

  • Rapid Miner has a responsive and intuitive GUI.

WEKA

About

Weka is java based open source data mining tool which has collection of data mining algorithms such as lazy, rules, decision trees and so on. It supports reading of files from several different databases. Weka also allows to import the data over internet, from web pages or from a remotely located SQL database server by entering the URL of resource.


General Features

  • Weka provides limited support to the visualization of concluded data. The representation of the results, graphs and plots is not in detail as compared to Rapid Miner.

  • Weka also provides available add-on functions to be included in order to be able to interface with R statistical package with improved statistical analysis and representation of result.

  • Weka provides three graphical user interfaces-
    A) The Explorer for exploratory data analysis to support preprocessing, attribute selection, learning, visualization.
    B) The Experimenter that provides experimental environment for testing and evaluating machine learning algorithms.
    C) The Knowledge Flow for new process evaluation.

  • It is suitable for developing new machine learning schemes.

  • Weka loads data file in formats of ARFF, CSV, C4.5, binary.

  • It is free, extensible and can be integrated into other java packages.

  • Weka is best suited for mining association rules.

  • Weka require less memory to install hence works faster.

TABLEAU

About

Tableau is a recent business intelligence tool used to easily visualize the data and help to solve complex analytical problems. It is a very poewrful tool used to generate reports and statistics. Can be used with R-package to implement data mining algorithms. Another important feature which has made Tableau popular among data analysists is its support to share the data on the cloud or use saved work remotely in a secure fashion.


General Features

  • Exploring and presenting your data is easier, faster, and more beautiful.

  • Tableau is extensively used in business intelligence

  • Tableau consists of a Dashboard which can be imported from anywhere

  • One of the other features of Tableau is that the whole process of implementing the projects and working with data is less time consuming and easy to handle.

  • Tableau provides support for large deployments of dashboards using Tableau Drive.

R

About

R is a popular open source statistical analysis tool. It is very similar to Matlab and uses same logic and user can easily load CSV format file and work with a number of functions. Rintegration can be used in combination with Tableau to create visual representations of various data mining algorithms. R mainly supports implementation of Naïve Bayes algorithm, confusion matrix and summary of the data.


General Features

  • R is primarily based on C and FORTRAN programming language.

  • It is a very popular tool for performing statistical analysis.

  • R can be integrated with Tableau using Rserve to get more information from your data using Tableau's easy drag-and-drop environment.

  • R implements operations using a command line interface . A large number of free "packages" are readily available for performing different data mining, machine learning and statistical techniques.

  • Although, R is less specialised towards data mining, it does support some of the data mining algorithms.

  • R involves a code-driven methodology which involves use of a number of in-built functions and commands to perform statistical analysis and data mining.

  • R is widely implemented on UNIX systems and extensively used for statistical computing.

  • For performing data mining, R uses WEKA preprocessing funtion as it does not have its own preprocessing function.
